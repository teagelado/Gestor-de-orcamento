# 💰 Gestor de Orçamento

Aplicação web para controle de finanças pessoais desenvolvida utilizando **HTML, CSS e JavaScript**.

O sistema permite adicionar ganhos e despesas, calcular automaticamente o saldo total e visualizar o histórico de transações.

---

# 🚀 Funcionalidades

* Adicionar novas transações
* Registrar **ganhos (valores positivos)**
* Registrar **despesas (valores negativos)**
* Cálculo automático de:

  * Balanço total
  * Total de lucros
  * Total de despesas
* Histórico de movimentações
* Interface simples e intuitiva

---

# 🛠️ Tecnologias Utilizadas

* **HTML5** → Estrutura da aplicação
* **CSS3** → Estilização da interface
* **JavaScript (Vanilla JS)** → Lógica da aplicação e manipulação do DOM

---

# 📷 Interface

A aplicação possui um painel central que mostra:

* **Balanço total**
* **Lucros**
* **Despesas**
* **Histórico de transações**

O usuário pode adicionar uma nova transação informando:

* Descrição
* Valor da transação

Valores **positivos representam ganhos**
Valores **negativos representam despesas**
